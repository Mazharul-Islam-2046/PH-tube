Scope Of Var, Let and Const

1. Scope of Var

=>Var is Function scoped, has hoisting ability and has no block scope.

2. Scope Of Let

=> Let has hoisting ability and also block scoped.

3. Scope Of Const

=> Const has no hoisting ability But block scoped.

Use Case Of Null and Undifined

=>People use null to fill the variable with no value. But Undifined represents something doesn't exist
